**Class: `IrrigationMonitorConfigFlow(config_entries.ConfigFlow, domain=DOMAIN)`**
- `VERSION = 1`
- Instance variable `_flume_entity_id: str` set in async_step_user

**Method: `async_step_user(self, user_input: dict | None = None)`**
- If user_input is not None: store `self._flume_entity_id = user_input[CONF_FLUME_ENTITY_ID]`, return `await self.async_step_valves()`
- Otherwise: show form with step_id="user", schema uses `EntitySelector(EntitySelectorConfig(filter=EntityFilterSelectorConfig(integration="flume")))` for CONF_FLUME_ENTITY_ID

**Method: `async_step_valves(self, user_input: dict | None = None)`**
- If user_input is not None: build `options_zones` dict where each selected entity_id maps to `{"shutoff_enabled": DEFAULT_SHUTOFF_ENABLED, "alerts_enabled": DEFAULT_ALERTS_ENABLED, "calibrated_flow": None, "threshold_multiplier": DEFAULT_THRESHOLD_MULTIPLIER}`. Then call `self.async_create_entry(title="Irrigation Monitor", data={CONF_FLUME_ENTITY_ID: self._flume_entity_id, CONF_MONITORED_ZONES: user_input[CONF_MONITORED_ZONES], CONF_POLL_INTERVAL: DEFAULT_POLL_INTERVAL}, options={CONF_ZONES: options_zones})`
- Otherwise: call `self._discover_valve_entities()` to get candidates, build SelectOptionDict list with `label=f"{name} ({eid})"`, show form with step_id="valves" using `SelectSelector(SelectSelectorConfig(options=options, multiple=True))`

**Method: `_discover_valve_entities(self) -> list[tuple[str, str]]`**
- Get registry via `er.async_get(self.hass)`
- Iterate `registry.entities.values()`, filter by `entry.domain in VALVE_DOMAINS`
- For each match: `name = entry.name or entry.original_name or entry.entity_id`
- Return sorted list of `(entity_id, name)` tuples sorted by name
