```python
    daily_usage: float  # gal since midnight
```

**Class: `IrrigationCoordinator(DataUpdateCoordinator[dict[str, ZoneData]])`:**

**`__init__`** — MUST pass `config_entry=entry` explicitly (HA 2026.x requirement):

```python
    hass,
    _LOGGER,
```

**`_async_setup`** — Store load + midnight registration:
- `stored = await self._store.async_load()`
- If `stored` and `stored.get("date") == date.today().isoformat()` => restore `self._daily_totals = dict(stored.get("zones", {}))`
- Otherwise => `self._daily_totals = {}` and `self._store.async_delay_save(self._data_to_save, 0)` (NOT await — `async_delay_save` is `@callback`)
- Register midnight reset: `unsub = async_track_time_change(self.hass, self._midnight_reset, hour=0, minute=0, second=0)` then `self._entry.async_on_unload(unsub)`

**`_async_update_data`** — core poll logic:
1. Read Flume state: `state = self.hass.states.get(flume_id)`
2. Guard: `if state is None or state.state in (STATE_UNAVAILABLE, STATE_UNKNOWN): raise UpdateFailed(f"Flume sensor {flume_id} is unavailable")`
3. Convert: `flume_flow = float(state.state)` in try/except ValueError => raise UpdateFailed
4. Get `zones_config = self._entry.options.get(CONF_ZONES, {})`
5. Get `monitored = self._entry.data[CONF_MONITORED_ZONES]`
6. Build active_calibrated dict: `{z: zones_config[z][CONF_CALIBRATED_FLOW] for z in monitored if self._zone_is_on(z) and zones_config.get(z, {}).get(CONF_CALIBRATED_FLOW) is not None}`
7. `calibrated_sum = sum(active_calibrated.values())`
8. For each zone in monitored:
   - If not on: flow_rate=0.0, usage_increment=0.0
   - If on AND calibrated AND in active_calibrated:
     - `threshold = zone_cfg.get(CONF_THRESHOLD_MULTIPLIER, 1.5)`
     - Multi-zone overlap check: `if calibrated_sum > 0 and abs(flume_flow - calibrated_sum) / calibrated_sum <= (threshold - 1.0): flow_rate = calibrated_flow`
     - Single calibrated zone: `elif len(active_calibrated) == 1: flow_rate = flume_flow` (attribute full Flume reading to single running zone)
     - Otherwise: `flow_rate = 0.0` (ambiguous overlap, skip)
   - If on AND uncalibrated: `flow_rate = flume_flow if len(active_calibrated) == 0 else 0.0` (full Flume only if no calibrated zones muddying it)
   - `usage_increment = flow_rate * (interval_seconds / 60.0)` — convert gal/min to gal for this interval
   - `self._daily_totals[zone_id] = self._daily_totals.get(zone_id, 0.0) + usage_increment`
   - Result: `ZoneData(flow_rate=flow_rate, daily_usage=self._daily_totals[zone_id], is_available=True)`
9. `self._store.async_delay_save(self._data_to_save, SAVE_DELAY)` — NOT await

**`_zone_is_on`** — CRITICAL: valve domain uses "open"/"closed", not "on"/"off":
```python
def _zone_is_on(self, entity_id: str) -> bool:
    state = self.hass.states.get(entity_id)
    if state is None:
        return False
    if entity_id.startswith("valve."):
        return state.state == "open"
    return state.state in ("on", "true")
```

**`_data_to_save`** — returns dict for Store:
```python
def _data_to_save(self) -> dict:
    return {"date": date.today().isoformat(), "zones": dict(self._daily_totals)}
```

**`_midnight_reset`** — async callback:

```python
async def _midnight_reset(self, _now) -> None:

    await self.async_refresh()
```
